# 총관리자 (Admin) 기능 가이드

> 총관리자는 시스템 전체를 관리하며, 모든 역할의 기능에 접근할 수 있습니다.

---

## 1. 로그인 후 첫 화면 (컨트롤 타워)

### 1.1 진행자 배정 탭 (기본)
- 연월브랜드 목록 표시 (접기/펼치기 가능)
- 캠페인 클릭 → 해당 캠페인의 품목별 진행자 배정
- 제품명 클릭 → 제품 상세 정보 팝업
  - 기본 정보, 키워드/구매 정보, 출고 정보
  - 상품 URL, 리뷰가이드, 비고

### 1.2 진행자/영업사/브랜드사 탭
- 각 역할의 사용자 목록 표시
- 사용자 선택 → 해당 사용자의 대시보드 미리보기
- "전체 화면 보기" 버튼 → 해당 사용자 대시보드 전체 조회

---

## 2. 진행자 배정 기능

### 2.1 신규 배정
- 품목별 드롭다운에서 진행자 선택
- "저장" 버튼 클릭 시 배정 완료
- 상태: `미배정` → `배정 완료`

### 2.2 진행자 재배정
- 이미 배정된 품목의 드롭다운에서 다른 진행자 선택
- 상태: `배정 완료` → `변경 중` (저장 전)
- "저장" 버튼 클릭 시 재배정 완료

### 2.3 배정 취소
- 드롭다운에서 "선택 안 함" 선택
- 상태: `배정 완료` → `취소 예정` (저장 전)
- "저장" 버튼 클릭 시 배정 취소

### 2.4 상태 칩 표시
| 상태 | 색상 | 설명 |
|-----|-----|-----|
| 배정 완료 | 녹색 | 현재 진행자가 배정됨 |
| 변경 중 | 주황 | 다른 진행자로 변경 대기 |
| 취소 예정 | 빨강 | 배정 취소 대기 |
| 미배정 | 회색 | 아직 배정되지 않음 |

## 3. 캠페인/품목 관리 (영업사 기능)

### 3.1 연월브랜드 추가
- 상단 "연월브랜드 추가" 버튼 클릭
- 연월 선택 + 브랜드 선택
- 조건: 동일 연월+브랜드 조합 불가

### 3.2 브랜드 추가
- 상단 "브랜드 추가" 버튼 클릭
- 브랜드명 입력
- 자동으로 현재 영업사에 할당

### 3.3 캠페인 추가
- 사이드바에서 연월브랜드 우클릭 또는 "+" 버튼
- 입력 항목:
  - 캠페인명
  - 설명 (선택)
  - 시작일/종료일 (선택)
  - 브랜드 연결
  - 상태: 신규/보류

### 3.4 품목 추가
- 캠페인 선택 후 "품목 추가" 버튼
- 입력 항목:
  - 제품명, 미출고/실출고
  - 희망 유입 키워드
  - 총 구매 건수
  - 일 구매 건수 (슬래시 구분 가능: "6/6" 또는 "2/2/2/...")
  - 상품 URL, 구매 옵션, 플랫폼
  - 상품 가격, 출고 마감 시간
  - 리뷰가이드 및 소구점
  - 택배대행 여부
  - 비고

## 4. 구매자 관리 (진행자 기능)

### 4.1 구매자 조회
- 캠페인 → 품목 선택 시 구매자 시트 표시
- Handsontable 기반 엑셀 형식

### 4.2 구매자 추가
- 단일 추가: 각 필드에 직접 입력
- 일괄 추가: 슬래시 구분 텍스트 붙여넣기
  - 형식: `주문번호/구매자/수취인/아이디/연락처/주소/계좌번호/금액`

### 4.3 구매자 수정/삭제
- 시트에서 직접 편집 후 Ctrl+S 또는 저장 버튼
- 행 선택 후 삭제

### 4.4 업로드 링크 복사
- 품목별 업로드 링크 복사 버튼
- 구매자에게 전달하여 이미지 업로드 유도

---

## 5. 입금 관리

### 5.1 품목별 입금 확인
- 캠페인 → 품목 선택 → 입금관리 버튼
- 구매자별 입금 완료 스위치 토글
- 실시간 반영 (새로고침 불필요)

### 5.2 날짜별 입금 관리 (헤더 버튼)
- 상단 "날짜별 입금관리" 버튼 클릭
- 달력에서 날짜 선택
- 해당 날짜의 전체 구매자 목록 표시
- 일괄 입금 확인 가능

---

## 6. 마진 현황

### 6.1 마진 대시보드 (헤더 버튼)
- 상단 "마진 현황" 버튼 클릭
- 캠페인별 매출/지출/마진 표시

### 6.2 지출 입력 (Admin 전용)
- 품목별 "지출 입력" 버튼 클릭
- 입력 항목:
  - 제품비
  - 택배비
  - 리뷰비용
  - 기타비용
- 마진 자동 계산: `총매출(VAT포함) - 총지출`

### 6.3 마진 계산 방식
- 총매출 = 총구매건수 × 공급가 × 1.1 (부가세)
- 마진율 = (마진 / 총매출) × 100%

---

## 7. 사용자 관리

### 7.1 사용자 등록 (헤더 버튼)
- 상단 "사용자 등록" 버튼 클릭
- 입력 항목:
  - 역할 선택: 총관리자/영업사/진행자/브랜드사
  - 이름
  - 사용자명 (로그인 ID)
  - 초기 비밀번호

### 7.2 비밀번호 초기화
- 사용자 목록에서 해당 사용자 선택
- "비밀번호 초기화" 버튼
- 임시 비밀번호 발급

### 7.3 사용자 비활성화/삭제
- 사용자 목록에서 해당 사용자 선택
- 비활성화: 로그인 차단 (데이터 유지)
- 삭제: 완전 삭제 (관련 데이터 영향 주의)

---

## 8. 로그인 페이지 설정 (헤더 버튼)

### 8.1 공지사항 설정
- 상단 "설정" 버튼 클릭
- 공지사항 텍스트 입력
- 로그인 화면에 표시됨

### 8.2 배너 제목 설정
- 로그인 화면 상단 배너 문구 변경

---

## 9. 송장 관리

### 9.1 송장번호 입력
- 캠페인 → 품목 → 구매자 시트
- 송장번호 컬럼에 직접 입력
- 저장 시 자동 반영

### 9.2 배송 지연 표시
- 배송 지연 체크박스 선택 시 표시

---

## 10. 알림 기능 (헤더 아이콘)

### 10.1 알림 확인
- 상단 알림 아이콘 (숫자 배지: 읽지 않은 알림 수)
- 클릭 시 알림 목록 표시
- 알림 클릭 → 해당 품목/캠페인으로 이동

### 10.2 모두 읽음 처리
- 알림 목록 상단 "모두 읽음" 버튼

---

## 11. 프로필 관리

### 11.1 프로필 수정
- 상단 프로필 아이콘 클릭 → "프로필 수정"
- 이름 변경
- 비밀번호 변경

### 11.2 로그아웃
- 상단 프로필 아이콘 클릭 → "로그아웃"
- 세션 완전 종료

---

## 12. 리뷰 현황 조회 (브랜드사 기능)

### 12.1 캠페인별 리뷰 진행률
- 사이드바에서 연월브랜드 → 캠페인 선택
- 진행률 퍼센트 표시
- 완료 시 녹색 체크 아이콘

### 12.2 구매자 리뷰 이미지 확인
- 제한된 컬럼만 표시 (주문번호, 구매자, 수취인, 아이디, 리뷰샷, 송장번호)
- 이미지 클릭 시 확대 팝업
