# 브랜드사 (Brand) 기능 가이드

> 브랜드사는 자사 캠페인의 리뷰 진행 현황을 모니터링합니다. 정보 보호를 위해 제한된 컬럼만 조회 가능합니다.

---

## 1. 로그인 후 첫 화면

### 1.1 화면 구성
- **좌측 사이드바**: 연결된 연월브랜드 → 캠페인 계층 구조
- **우측 메인**: 선택한 캠페인의 리뷰 현황 시트
- **상단 헤더**: 알림, 프로필

### 1.2 사이드바 표시 정보
- 연월브랜드명 (예: "2601오브제")
- 캠페인별 진행률 표시:
  - **100% 완료**: 녹색 체크 아이콘
  - **진행 중**: 퍼센트 칩 (색상별 구분)
- 캠페인 상태 칩: 신규 / 진행 / 완료

### 1.3 진행률 계산 기준
- **분모**: 전체 등록된 구매자 수
- **분자**: 리뷰 이미지 업로드 완료된 구매자 수
- 조건: 선 업로드(임시 구매자)는 제외

---

## 2. 캠페인 조회

### 2.1 캠페인 목록
- 사이드바에서 연월브랜드 클릭 → 하위 캠페인 펼침
- 연결된 캠페인만 표시

### 2.2 캠페인 상태
| 상태 | 색상 | 설명 |
|-----|-----|-----|
| 신규 | 노랑 | 새로 등록된 캠페인 |
| 진행 | 파랑 | 리뷰 진행 중 |
| 완료 | 녹색 | 모든 리뷰 완료 |

### 2.3 숨기기 기능
- 캠페인 우측 메뉴 → "숨기기"
- "숨겨진 항목 보기" 토글로 복원

---

## 3. 리뷰 현황 조회

### 3.1 조회 가능 컬럼 (제한됨)
| 컬럼 | 설명 |
|-----|-----|
| 주문번호 | 구매자의 주문번호 |
| 구매자 | 구매자명 |
| 수취인 | 수취인명 |
| 아이디 | 구매자 아이디 |
| 리뷰샷 | 업로드된 리뷰 이미지 |
| 송장번호 | 배송 송장번호 |

### 3.2 숨겨진 컬럼 (보안)
- 연락처
- 주소
- 계좌정보
- 금액
- 입금여부

### 3.3 선 업로드 데이터
- 조건: 브랜드사에게는 표시되지 않음
- 정상 등록된 구매자만 조회 가능

---

## 4. 이미지 조회

### 4.1 테이블 뷰
- 리뷰샷 컬럼에 썸네일 표시
- 클릭 시 이미지 확대 팝업

### 4.2 갤러리 뷰
- 이미지 상세 정보 표시:
  - 주문번호
  - 구매자명
  - 송장번호
- 이미지 확대/축소 가능

### 4.3 이미지 저장
- 이미지 우클릭 → "이미지 저장" (브라우저 기능)

---

## 5. 알림 기능 (헤더 아이콘)

### 5.1 알림 확인
- 상단 알림 아이콘 (숫자 배지: 읽지 않은 알림 수)
- 클릭 시 알림 목록 표시

### 5.2 알림 종류
- 새 캠페인 연결 알림
- 리뷰 완료 알림
- 기타 시스템 알림

### 5.3 모두 읽음 처리
- 알림 목록 상단 "모두 읽음" 버튼

---

## 6. 프로필 관리

### 6.1 프로필 수정
- 상단 프로필 아이콘 클릭 → "프로필 수정"
- 이름 변경
- 비밀번호 변경

### 6.2 로그아웃
- 상단 프로필 아이콘 클릭 → "로그아웃"

---

## 7. Admin 모드 (Admin이 대신 조회 시)

### 7.1 표시
- Admin이 브랜드사 대시보드 조회 시 상단에 "Admin으로 돌아가기" 버튼 표시

### 7.2 기능 동일
- 브랜드사와 동일한 제한된 뷰 표시
- Admin이라도 브랜드사 화면에서는 민감 정보 숨김

---

## 8. 제한 사항

### 8.1 조회만 가능
- 구매자 추가/수정/삭제 불가
- 캠페인/품목 수정 불가

### 8.2 정보 접근 제한
- 민감한 개인정보 (연락처, 주소, 계좌) 조회 불가
- 입금 관련 정보 조회 불가

### 8.3 업로드 링크 접근 불가
- 업로드 링크 복사 기능 없음
