# 진행자 (Operator) 기능 가이드

> 진행자는 배정받은 품목의 구매자(리뷰어) 정보를 관리하고, 이미지 업로드 링크를 공유합니다.

---

## 1. 로그인 후 첫 화면

### 1.1 화면 구성
- **좌측 사이드바**: 배정받은 연월브랜드 → 캠페인 계층 구조
- **우측 메인**: 선택한 캠페인의 품목 시트 (Handsontable)
- **상단 헤더**: 메모장, 선 업로드 알림, 프로필

### 1.2 사이드바 표시 정보
- 연월브랜드명 (예: "2601오브제")
- 캠페인별 진행률 표시:
  - **100% 완료**: 녹색 체크 아이콘
  - **80% 이상**: 녹색 배경 퍼센트 칩
  - **50~79%**: 노랑 배경 퍼센트 칩
  - **50% 미만**: 주황 배경 퍼센트 칩
- 신규 배정 품목: 주황색 "신규" 칩
- 구매자 없음 경고: 빨강 경고 아이콘

### 1.3 배정 상태 구분
- **당일 배정** = "신규" 표시
- **다음날 이후** = "진행" 표시 + 구매자 없으면 경고

---

## 2. 메모장 기능 (헤더 버튼)

### 2.1 메모장 열기
- 상단 "메모장" 버튼 클릭
- 개인 메모 작성/조회

### 2.2 자동 저장
- 다이얼로그 닫기 시 자동 저장
- 다음 로그인 시 복원

---

## 3. 선 업로드 알림 (헤더 아이콘)

### 3.1 알림 표시
- 상단 알림 아이콘 (숫자 배지: 선 업로드된 품목 수)
- 조건: 구매자 등록 전 이미지가 먼저 업로드된 경우

### 3.2 알림 클릭
- 클릭 시 선 업로드된 품목 목록 메뉴 표시
- 품목 클릭 → 해당 품목으로 이동

---

## 4. 품목 시트 (Handsontable)

### 4.1 컬럼 순서
| 순서 | 컬럼명 | 설명 |
|-----|------|-----|
| 1 | 접기 | 그룹 접기/펼치기 버튼 (20px) |
| 2 | 날짜 | 배정된 날짜 |
| 3 | 순번 | 일차별 순번 |
| 4 | 제품명 | 품목 이름 |
| 5 | 옵션 | 구매 옵션 |
| 6 | 플랫폼 | 쿠팡/네이버 등 |
| 7 | 출고 | 미출고/실출고 |
| 8 | 키워드 | 희망 유입 키워드 |
| 9 | 가격 | 상품 가격 |
| 10 | 총건수 | 총 구매 건수 |
| 11 | 일건수 | 일 구매 건수 |
| 12 | 택배 | 택배대행 여부 |
| 13 | URL | 상품 URL |
| 14 | 특이사항 | 리뷰가이드 등 통합 |

### 4.2 시트 기능
- **그룹 접기/펼치기**: 접기 컬럼 클릭
- **필터링**: 컬럼 헤더 호버 시 필터 버튼 표시
- **드래그 복사**: 셀 선택 후 드래그
- **복사/붙여넣기**: Ctrl+C / Ctrl+V
- **저장**: Ctrl+S 또는 저장 버튼
- **컬럼 너비 조정**: 컬럼 경계 드래그 (자동 저장)
- **스크롤**: Shift+마우스휠 = 횡스크롤

### 4.3 건수/금액 합계
- 필터링/접기 상태와 무관하게 전체 데이터 기준 합계 표시
- 시트 하단에 총 건수, 총 금액 표시

## 5. 구매자 추가

### 5.1 단일 추가
- 시트 빈 행에 직접 입력
- 입력 컬럼:

| 컬럼 | 필수 | 설명 |
|-----|-----|-----|
| 주문번호 | △ | 주문번호 또는 계좌번호 중 하나 필수 |
| 구매자 | - | 구매자명 |
| 수취인 | - | 수취인명 |
| 아이디 | - | 구매자 아이디 |
| 연락처 | - | 전화번호 |
| 주소 | - | 배송 주소 |
| 계좌번호 | △ | 계좌 정보 (주문번호 없을 시 필수) |
| 금액 | - | 구매 금액 |

### 5.2 일괄 추가 (슬래시 파싱)
- 주문번호 컬럼에 슬래시 구분 데이터 붙여넣기
- 형식: `주문번호/구매자/수취인/아이디/연락처/주소/계좌번호/금액`
- 자동으로 8개 컬럼에 분배

### 5.3 슬래시 파싱 예시
```
8100156654664/김민형/김민형/p4che@naver.com/010-8221-1864/경남 거제시 장승포동 123/부산112-2323-738601 김민지/22800
```
→ 자동 분리되어 각 컬럼에 입력

### 5.4 다중 행 일괄 추가
- 여러 줄 데이터 한번에 붙여넣기 가능
- 각 줄이 하나의 구매자로 추가

---

## 6. 구매자 수정

### 6.1 직접 편집
- 시트에서 해당 셀 클릭 후 수정
- Enter 또는 Tab으로 다음 셀 이동

### 6.2 저장
- Ctrl+S 또는 상단 "저장" 버튼
- 변경사항 자동 감지 후 저장

---

## 7. 구매자 삭제

### 7.1 행 삭제
- 삭제할 행 선택
- 삭제 버튼 클릭 또는 우클릭 메뉴
- 확인 후 삭제

### 7.2 조건
- 연결된 이미지가 있어도 삭제 가능
- 삭제된 구매자의 이미지는 "선 업로드" 상태로 변경

---

## 8. 업로드 링크 공유

### 8.1 링크 복사
- 품목 행 선택 → "업로드 링크 복사" 버튼
- 또는 품목 우클릭 → "업로드 링크 복사"

### 8.2 링크 형식
- URL: `https://your-domain.com/upload-slot/{토큰}`
- 토큰: 품목별 고유 UUID

### 8.3 사용 방법
- 구매자에게 링크 전달
- 구매자가 주문번호/계좌번호 입력 + 이미지 업로드
- 자동으로 구매자와 매칭

---

## 9. 이미지 확인

### 9.1 리뷰샷 컬럼
- 이미지 업로드 완료 시 썸네일 표시
- 클릭 시 이미지 확대 팝업

### 9.2 선 업로드 표시
- 구매자 등록 전 이미지 업로드된 경우
- 노란색 배경 + "선 업로드" 칩
- 구매자 등록 시 자동 매칭

### 9.3 이미지 매칭 우선순위
1. 주문번호 일치
2. 계좌번호 일치 (정규화 후 비교)

---

## 10. 입금명 수정

### 10.1 입금명 변경
- 구매자 시트에서 "입금명" 컬럼 직접 편집
- 조건: 입금명만 수정 가능 (입금확인 토글은 Admin만)

---

## 11. 그룹 관리

### 11.1 일차별 그룹
- 일 구매건수에 따라 자동 그룹 생성
- 예: "6/6" → 1일차 그룹(6건) + 2일차 그룹(6건)

### 11.2 그룹 접기/펼치기
- 접기 버튼 클릭 → 해당 그룹 숨김
- 다시 클릭 → 펼침

### 11.3 그룹 삭제
- 그룹 전체 삭제 시 하위 구매자도 삭제
- 경고 확인 후 삭제

---

## 12. 중복 주문번호 확인

### 12.1 자동 하이라이팅
- 동일 주문번호 입력 시 빨간색 배경 표시
- 데이터 정합성 확인용

### 12.2 확인 방법
- 시트에서 빨간색 배경 셀 확인
- 중복 주문번호 수정 또는 삭제

---

## 13. 숨기기 기능

### 13.1 캠페인 숨기기
- 캠페인 우측 메뉴 → "숨기기"
- 사이드바에서 숨겨짐 (데이터 유지)

### 13.2 연월브랜드 숨기기
- 연월브랜드 우측 메뉴 → "숨기기"
- 하위 캠페인 모두 숨겨짐

### 13.3 숨겨진 항목 보기
- 사이드바 상단 "숨겨진 항목 보기" 토글
- 토글 ON 시 숨겨진 항목 표시 (회색)

---

## 14. 프로필 관리

### 14.1 프로필 수정
- 상단 프로필 아이콘 클릭 → "프로필 수정"
- 이름 변경
- 비밀번호 변경

### 14.2 로그아웃
- 상단 프로필 아이콘 클릭 → "로그아웃"

---

## 15. Admin 모드 (Admin이 대신 조회 시)

### 15.1 표시
- Admin이 진행자 대시보드 조회 시 상단에 "Admin으로 돌아가기" 버튼 표시

### 15.2 기능
- Admin이 진행자 대신 구매자 추가/수정/삭제 가능
- Admin이 업로드 링크 복사 가능
