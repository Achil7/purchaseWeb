# 영업사 (Sales) 기능 가이드

> 영업사는 브랜드를 등록하고, 캠페인과 품목을 생성하여 관리합니다.

---

## 1. 로그인 후 첫 화면

### 1.1 화면 구성
- **좌측 사이드바**: 연월브랜드 → 캠페인 계층 구조
- **우측 메인**: 선택한 캠페인의 품목 시트 (Handsontable)
- **상단 헤더**: 브랜드 추가, 연월브랜드 추가, 마진 현황, 프로필

### 1.2 사이드바 기능
- 연월브랜드 클릭 → 펼치기/접기
- 캠페인 클릭 → 해당 캠페인의 품목 시트 표시
- 캠페인 상태 칩: 신규(노랑) / 진행(파랑) / 완료(녹색) / 취소(빨강)
- 숨겨진 항목 보기/숨기기 토글

---

## 2. 브랜드 등록

### 2.1 신규 브랜드 추가
- 상단 "브랜드 추가" 버튼 클릭
- 브랜드명 입력
- 저장 시 자동으로 본인에게 할당

### 2.2 조건
- 동일한 브랜드명 중복 불가
- 본인이 등록한 브랜드만 사용 가능

---

## 3. 연월브랜드 관리

### 3.1 연월브랜드 추가
- 상단 "연월브랜드 추가" 버튼 클릭
- 입력 항목:
  - 연월 선택 (예: 2601 = 2026년 1월)
  - 브랜드 선택 (본인이 등록한 브랜드 목록)
- 조건: 동일 연월+브랜드 조합 불가

### 3.2 연월브랜드 표시
- 사이드바에 연월브랜드명 표시 (예: "2601오브제")
- 클릭하면 하위 캠페인 목록 펼침

### 3.3 연월브랜드 숨기기
- 연월브랜드 우측 메뉴 → "숨기기"
- "숨겨진 항목 보기" 토글로 복원 가능

---

## 4. 캠페인 관리

### 4.1 캠페인 추가
- 연월브랜드 선택 후 "+" 버튼 또는 우클릭 메뉴
- 입력 항목:
  - **캠페인명** (필수)
  - 설명 (선택)
  - 시작일 (선택)
  - 종료일 (선택)
  - **상태**: 신규 / 보류 (기본값: 신규)

### 4.2 캠페인 상태
| 상태 | 색상 | 설명 |
|-----|-----|-----|
| 신규 | 노랑 | 새로 등록된 캠페인 |
| 진행 | 파랑 | 진행자 배정 후 진행 중 |
| 보류 | 회색 | 일시 중단 |
| 완료 | 녹색 | 모든 리뷰 완료 |
| 취소 | 빨강 | 취소된 캠페인 |

### 4.3 캠페인 수정
- 캠페인명 우측 편집 아이콘 클릭
- 캠페인명, 상태, 기간 수정 가능

### 4.4 캠페인 삭제
- 캠페인 우측 삭제 아이콘 클릭
- 조건: 하위 품목이 없어야 삭제 가능
- 경고 확인 후 삭제

### 4.5 캠페인 숨기기
- 캠페인 우측 메뉴 → "숨기기"
- 사이드바에서 숨겨짐 (데이터 유지)

---

## 5. 품목 관리

### 5.1 품목 추가
- 캠페인 선택 후 시트 상단 "품목 추가" 버튼
- 입력 항목:

| 필드 | 필수 | 설명 |
|-----|-----|-----|
| 제품명 | - | 제품 이름 |
| 미출고/실출고 | - | 출고 유형 구분 |
| 희망 유입 키워드 | - | 검색 키워드 |
| 총 구매 건수 | **O** | 전체 구매 목표 건수 |
| 일 구매 건수 | **O** | 하루 구매 건수 (슬래시 구분 가능) |
| 상품 URL | - | 제품 상세 페이지 링크 |
| 구매 옵션 | - | 구매 시 선택 옵션 |
| 플랫폼 | - | 쿠팡/네이버 등 |
| 상품 가격 | - | 단가 |
| 출고 마감 시간 | - | 당일 출고 기준 시간 |
| 리뷰가이드 | - | 리뷰 작성 가이드라인 |
| 소구점 | - | 강조 포인트 |
| 택배대행 | - | Y/N |
| 비고 | - | 추가 메모 |

**조건**: 총 구매 건수와 일 구매 건수의 합계가 일치해야 함

### 5.2 일 구매 건수 입력 방식
- 단일 값: `6` (매일 6건)
- 슬래시 구분: `6/6` (1일차 6건, 2일차 6건)
- 다중 일차: `2/2/2/2/2` (5일간 각 2건씩)

### 5.3 품목 수정
- 시트에서 직접 편집
- Ctrl+S 또는 저장 버튼으로 저장

### 5.4 품목 삭제
- 품목 행 선택 후 삭제 버튼
- 조건: 하위 구매자가 없어야 삭제 가능
- 경고 확인 후 삭제

---

## 6. 품목 시트 (Handsontable)

### 6.1 표시 컬럼
- 날짜, 순번, 제품명, 옵션, 플랫폼
- 출고, 키워드, 가격
- 총건수, 일건수, 택배
- 상품URL, 특이사항, 비고

### 6.2 시트 기능
- **드래그 복사**: 셀 선택 후 드래그
- **복사/붙여넣기**: Ctrl+C / Ctrl+V
- **컬럼 너비 조정**: 컬럼 경계 드래그 (자동 저장)
- **필터링**: 컬럼 헤더 호버 시 필터 버튼 표시
- **스크롤**: Shift+마우스휠 = 횡스크롤

### 6.3 중복 주문번호 표시
- 동일 주문번호 존재 시 빨간색 배경 하이라이팅
- 데이터 정합성 확인용

---

## 7. 구매자 조회 (읽기 전용)

### 7.1 구매자 목록 확인
- 시트에서 품목 클릭 → 구매자 정보 표시
- 조건: 조회만 가능 (수정/삭제 불가)

### 7.2 조회 컬럼
- 주문번호, 구매자, 수취인, 아이디
- 연락처, 주소, 계좌정보, 금액
- 입금여부, 리뷰샷

---

## 8. 입금명 수정

### 8.1 입금명 변경
- 구매자 시트에서 "입금명" 컬럼 직접 편집
- 조건: 입금명만 수정 가능 (다른 정보는 불가)

---

## 9. 마진 현황 조회

### 9.1 마진 대시보드 (헤더 버튼)
- 상단 "마진 현황" 버튼 클릭
- 본인이 생성한 캠페인만 표시

### 9.2 표시 항목
| 항목 | 설명 |
|-----|-----|
| 총매출 | 총구매건수 × 공급가 × 1.1 |
| 총지출 | Admin이 입력한 지출 합계 |
| 마진 | 총매출 - 총지출 |
| 마진율 | (마진 / 총매출) × 100% |

### 9.3 조건
- 지출 입력은 Admin만 가능
- Sales는 조회만 가능

---

## 10. 프로필 관리

### 10.1 프로필 수정
- 상단 프로필 아이콘 클릭 → "프로필 수정"
- 이름 변경
- 비밀번호 변경

### 10.2 로그아웃
- 상단 프로필 아이콘 클릭 → "로그아웃"

---

## 11. Admin 모드 (Admin이 대신 조회 시)

### 11.1 표시
- Admin이 영업사 대시보드 조회 시 상단에 "Admin으로 돌아가기" 버튼 표시

### 11.2 기능
- Admin이 영업사 대신 브랜드/연월브랜드/캠페인/품목 생성 가능
- 생성된 항목은 해당 영업사 소유로 저장
