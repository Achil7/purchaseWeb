# 올리브영 랭킹 자동 수집기 - 사용 안내서

> **컴퓨터를 잘 모르셔도 따라 할 수 있게 작성했습니다.**
> 막히는 부분이 있으면 개발자에게 화면 캡처와 함께 연락해주세요.

---

## 처음 한 번만 하는 셋업 (약 20~30분)

이미 셋업이 끝났으면 [매번 사용법](#매번-사용법-3분)으로 가세요.

### A. 받아야 할 파일 (총 3개)

개발자에게서 받는 파일:

| 파일 | 설명 | 파일 이름 |
|------|------|-----------|
| 1️⃣ 프로그램 압축 파일 | 카카오톡/구글로 받음 | `purchaseweb.zip` (또는 비슷한 이름) |
| 2️⃣ 비밀번호 설정 파일 | 카카오톡으로 받음 | `.env` |
| 3️⃣ 서버 접속 키 | 카카오톡으로 받음 | `server_rsa_key.pem` |

3개 모두 받았는지 먼저 확인하세요.

---

### B. Step 1. 프로그램 압축 풀기

1. 받은 `purchaseweb.zip` 파일을 **바탕화면**에 두세요.
2. zip 파일을 **마우스 우클릭 → "압축 풀기"** 클릭
3. 압축 풀린 폴더 이름이 `purchaseweb` 인지 확인
   - 만약 `purchaseweb-main` 같은 이름이면 → `purchaseweb` 으로 이름 변경

이제 바탕화면에 `C:\Users\사용자이름\Desktop\purchaseweb` 폴더가 있으면 OK.

---

### C. Step 2. Node.js 설치

1. 인터넷 브라우저 (크롬/엣지) 열기
2. 주소창에 `https://nodejs.org/ko` 입력하고 Enter
3. 화면 가운데 **큰 초록색 LTS 버튼** 클릭 → 설치 파일 다운로드
4. 다운로드 완료된 설치 파일 (`node-vXX.XX.XX-x64.msi`) 더블클릭
5. 설치 화면이 뜨면 모든 단계 **"Next" 또는 "다음"** 클릭하면서 끝까지 진행
6. 마지막에 **"Finish" 또는 "마침"** 클릭

설치 확인 (선택):
- `Win + R` 키 → `cmd` 입력 → Enter
- 검은 창에 `node -v` 입력 → `v22.XX.XX` 같은 게 나오면 OK
- 창 닫기

---

### D. Step 3. 설정 파일 두기

받은 `.env` 파일과 `server_rsa_key.pem` 파일을 정해진 위치에 두어야 합니다.

#### `.env` 파일 위치

`C:\Users\사용자이름\Desktop\purchaseweb\backend\` 폴더 안에 두세요.

방법:
1. 바탕화면의 `purchaseweb` 폴더 더블클릭으로 열기
2. 안에 있는 `backend` 폴더 더블클릭으로 열기
3. 받은 `.env` 파일을 그 폴더 안으로 끌어다 놓기

#### `server_rsa_key.pem` 파일 위치

`C:\Users\사용자이름\.ssh\` 폴더 안에 두세요.

방법:
1. 키보드에서 `Win + R` 동시에 누르기
2. 작은 창이 뜨면 `%USERPROFILE%\.ssh` 입력 → Enter
   - 만약 "찾을 수 없음" 오류가 뜨면: 다음 셋업 단계에서 자동으로 폴더가 만들어지니 일단 건너뛰고, 그때 다시 두기
3. 폴더가 열리면 받은 `server_rsa_key.pem` 파일을 그 폴더 안에 끌어다 놓기

---

### E. Step 4. 자동 셋업 실행

1. 바탕화면 → `purchaseweb` 폴더 → `backend` 폴더 → `scripts` 폴더 열기
2. **`setup.bat` 파일 더블클릭**
3. 검은 창이 뜨면서 자동으로 진행됩니다 (5~15분 소요)
4. 화면 안내대로 따라가시면 됩니다
5. 마지막에 **"✅ 셋업 완료!"** 가 보이면 끝

도중에 오류 메시지가 뜨면:
- ❌ 빨간색 메시지의 안내대로 조치
- 그래도 모르면 화면 캡처해서 개발자에게 전달

---

### F. Step 5. 절전 모드 끄기 (중요)

수집 중에 컴퓨터가 절전 모드에 들어가면 데이터 수집이 멈춥니다. 끄셔야 합니다.

1. `Win + I` 동시에 누르기 (윈도우 설정 열림)
2. **시스템** 클릭
3. **전원** (또는 "전원 및 배터리") 클릭
4. **"화면 끄기"** : 원하는 시간 (10~30분 등) 선택
5. **"절전 모드로 전환"** : **"안 함"** 선택 ← 이게 핵심!

## 매번 사용법 (3분)

셋업이 끝나면 매일 이렇게만 하면 됩니다.

### 1. 수집 시작

1. 바탕화면 → `purchaseweb` 폴더 → `backend` 폴더 → `scripts` 폴더 열기
2. **`runRankingWorker.bat` 파일 더블클릭**
3. 검은 창이 뜨면 화면 안내대로 입력:

   ```
   [1] test  (DB: serverdb_test)
   [2] main  (DB: serverdb)
   번호 입력 (1 또는 2): 2
   ```
   → 보통 `2` (main, 운영) 선택. 안전 확인이 한 번 더 나오면 `y` 입력.

4. 그 다음 4가지 입력:

   | 항목 | 입력 예시 | 설명 |
   |------|----------|------|
   | 시작일 | (그냥 Enter) | "지금 이 시각"부터 시작 |
   | 종료일 | `2026-05-31` | 언제까지 수집할지 |
   | 인터벌 | `30` (또는 그냥 Enter = 20분) | 몇 분마다 수집할지 |
   | 시작 확인 | `y` | 시작 |

5. 자동으로 수집이 시작됩니다.

### 2. 작은 창 (ranking-ssh-tunnel) 신경 쓰지 마세요

- 검은 작은 창이 하나 더 떴을 거예요 (제목: ranking-ssh-tunnel)
- **이 창은 절대 닫지 마세요!**
- 데이터를 서버로 보내는 통로입니다. 닫으면 저장이 안 됩니다.
- 그냥 두면 됩니다 (최소화 상태로 둬도 OK)

### 3. 그 다음에 할 일

- ✅ 두 검은 창은 그대로 두세요
- ✅ 컴퓨터를 다른 일에 사용하셔도 됩니다
- ❌ 컴퓨터를 끄거나 절전 모드 X
- ❌ 인터넷 끊기지 않게 주의

### 4. 종료 방법

| 방법 | 설명 |
|------|------|
| **Ctrl + C** | 메인 창에서. 진행 중인 라운드 마무리 후 정상 종료 (가장 안전) |
| 자동 종료 | 입력한 종료일에 도달하면 자동 종료 |
| 창 닫기 | 강제 종료 (급할 때만) |

## 문제 해결

### 셋업 단계에서 `setup.bat`이 그냥 사라져요

화면이 너무 빨리 닫혀서 메시지를 못 봤을 수 있어요. 다음과 같이 시도:

1. `Win + R` → `cmd` → Enter
2. 검은 창에 다음 입력 (한 줄씩):
   ```
   cd %USERPROFILE%\Desktop\purchaseweb\backend\scripts
   setup.bat
   ```
3. 이번에는 메시지가 멈춰있을 거예요. 화면 캡처해서 개발자에게 전달.

### 수집 시작하니까 "DB 연결 실패"가 떠요

- ranking-ssh-tunnel 창이 떠있는지 확인 (작업표시줄)
- 인터넷 연결 확인
- 그래도 안 되면 개발자 연락 (네트워크 환경 설정 변경 가능성)

### 컴퓨터 재부팅했어요

다시 `runRankingWorker.bat` 더블클릭해서 처음부터 시작하세요. 다음 정각부터 자동으로 다시 수집해요.

### 토스트 알림이 안 떠요

- 윈도우 설정 → 시스템 → 알림 → "이 앱에서 알림 받기" 켜기

### 절전 모드를 깜빡 안 끄고 잠들었어요

- 다시 `runRankingWorker.bat` 시작하면 그 시점부터 다시 수집됩니다
- 그동안의 데이터는 누락됩니다

### "검은 창이 바로 꺼져요" / 알 수 없는 오류

`backend\logs\` 폴더 안에서 가장 최근 `.log` 파일을 찾아 개발자에게 전달.

## 폴더 구조 참고

```
바탕화면/
└─ purchaseweb/                              ← 받은 압축 풀린 폴더
   └─ backend/
      ├─ .env                                ← 받은 .env 여기에
      ├─ logs/                               ← 자동 생성, 로그 저장됨
      └─ scripts/
         ├─ setup.bat                        ← 처음 한 번만 더블클릭
         ├─ runRankingWorker.bat             ← 매번 더블클릭
         └─ RANKING_WORKER_README.md         ← 이 파일

C:/Users/사용자이름/.ssh/
└─ server_rsa_key.pem                        ← 받은 키 여기에
```
